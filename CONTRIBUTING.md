# 🤝 Contributing to Linux Hardening Tool

Thank you for your interest in contributing!

---

## 🚀 How to Contribute

1. Fork the repository
2. Create a feature branch:
   ```bash
   git checkout -b feature-name
   ```
3. Commit changes:
   ```bash
   git commit -m "feat: add new security feature"
   ```
4. Push and create a Pull Request

---

## 🔐 Contribution Guidelines

- Keep security first (no unsafe defaults)
- Follow Linux best practices
- Maintain backward compatibility
- Test on Ubuntu/Debian/RHEL environments

---

## 🧠 Suggested Contributions

- Telegram alert integration
- GeoIP tracking for attackers
- SIEM integration (Wazuh / ELK)
